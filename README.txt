# PassFort


## Backend Development Setup

To set up the backend development environment, you'll need the **Pipenv** Python package.

### Steps:

1. Install Pipenv:
    Run the following command in your shell:
    'pip install pipenv'

2. Navigate to the 'Backend' Directory

3. Run the following command to install all necessary dependencies listed in the Pipfile and Pipfile.lock:
    'pipenv install'

4. Start the virtual environment by running:
    'pipenv shell'




Additional Information:

You can check what dependencies you have installed by running:
    'pip freeze'
